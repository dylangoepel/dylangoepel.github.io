Finding orthonormal bases of subspaces

Exercises

Let \(U := \{ (a, b, c, d) \in \mathbb{R}^4 \mid a + b + c + d = 0 \}\). Find orthonormal bases of \(U\) and \(U^\bot\).

Solutions

To find an orthonormal base of \(U\), first notice that \(B = (b_1, b_2, b_3) := ((1, 0, 0, -1)^t, (0, 1, 0, -1)^t, (0, 0, 1, -1)^t)\) is a base of \(U\). Now, apply the Gram-Schmidt orthonormalization algorithm

\[ u_1 = (2^{-1/2}, 0, 0, -2^{-1/2}) \\ u_2 = (-6^{-1/2}, 2 \cdot 6^{-1/2}, 0, -6^{-1/2})^t \\ u_3 = (66^{-1/2}, -2 \cdot 66^{-1/2}, 6 \cdot 66^{-1/2}, -5 \cdot 66^{-1/2})^t \]

As \(U^\bot\) is one dimensional, finding a orthonormal base is trivial. Take for example

\[ B = ((4^{-1/2}, 4^{-1/2}, 4^{-1/2}, 4^{-1/2})^t) \]
